# Usage

## Install emacs (Arch)

```bash
yay -S emacs
```

## Install configs

```bash
git clone https://github.com/ccmikechen/.emacs.d.git <config-path>
ln -s <config-path> ~/.emacs.d
```

## Install roswell

```bash
yay -S roswell
```

## Install roswell slime plugin

```bash
ros install slime
```

## Run emacs then done!
